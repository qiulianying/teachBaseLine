Bootstrap Migration

讲解以及这个破玩意的坑!

jquery semantici ui：据说这个东东非常的好用，我感觉和bootstrap其实差不多，作为一个jquery的ui框架而已； 而且这个对移动端有做样式的兼容和适配

安装：
官网
根据官网的做法，通过以下方式进行安装
npm install semantic-ui --save
cd semantic/
gulp build

但是实际上我们按照第一步的时候，发现这个破玩意会卡到一直卡在Creating src/theme.config (less config) src/theme.config这一步。找到了解决方法，方法就是切换到特定的 NodeJS 版本

  1. 首先按照nvm的node管理工具，然后切换node版本：下载地址
  2. 主要：目前发现可以接受的node版本是v10.1.0
